DataHub 매물연동 API (1.0.0)

Download OpenAPI specification:

KB차차차의 DataHub 매물연동 API입니다. 차량 매물 등록, 수정, 공통코드 조회 등의 기능을 제공합니다.

⚠️ 인증정보는 heeseung.lee@kbfg.com, yulimlee@kbfg.com로 문의

⚠️ 신규시스템 (DataHub) 이용 시 주의사항:

  1. 매물수정
  • 기존 시스템으로 등록한 매물은 수정이 불가합니다.
  • 기존 매물은 판매완료 처리 후, 신규 매물로 등록한 뒤 이 API를 통해 수정해야 합니다.
  1. 판매완료
  • 기존 시스템 매물도 이 API를 통해 판매완료 처리 가능합니다.
  • param_gbn: car_no, param_value: 차량번호로 호출 시, KB차차차에 동일 partner_code로 등록된 차량에 대해 처리됩니다.
  • ⚠️ 단, DataHub에 등록되지 않은 차량번호에 대한 판매완료 처리는 일정 기간만 허용됩니다.
  • 일반적인 DataHub 등록 매물의 경우 param_gbn: partner_car_seq를 이용해야 합니다.

판매등록

판매등록

중요 안내 (신규시스템 DataHub)

  1. 판매완료
    기존 시스템으로 등록한 매물은 판매완료 신규 API 이용 가능하며,
    param_gbn에 car_no, param_value에 차량번호 입력 시 KB차차차에 동일 차량번호와 partner_code로 등록된 매물에 대해 판매완료 처리됩니다.
    (* datahub에 등록되지 않은 차량번호를 이용한 KB차차차 판매완료처리는 일정기간동안만 처리 예정)
    (* 일반적인 datahub 이용된 매물의 경우 param_gbn에 partner_car_seq 이용해야 합니다.)
